# Galaga 技術規格書：03_關卡與敵人設計 (Level & Enemy Design)

## 1. 過關技術流程 (Stage Transition Flow)
系統需透過全域狀態機 (Global State Machine) 嚴格控制關卡轉換，確保視覺與邏輯同步：

1.  **敵機清空偵測 (Enemies Zero Check)**:
    - 條件：`FormationCount == 0` 且 `ActiveSpriteCount == 0`。
    - 延遲：最後一名敵機爆炸動畫結束後等待 60 幀 (1s)。
2.  **階段結算 (Stage Bonus)**:
    - 非挑戰關卡：僅播放過關簡短音效。
    - **挑戰關卡 (Challenging Stage)**:
        - 暫停星空背景滾動。
        - 顯示 "Number of Hits" 與 "Bonus Points"。
        - 播放結算音樂。
3.  **標誌更新 (Icon Update)**:
    - 畫面右下角更新「關卡標誌」。
    - 規則：5 關為一個「五角星別針」，1 關為一個「藍紅條紋」。
4.  **下一關初始化 (Next Stage Init)**:
    - 重置敵機池 (Enemy Pool)。
    - 根據 `StageNumber` 計算基礎 `Rank` 加值。
    - 觸發第一組 (Batch 1/5) 進場程序。

## 2. 30 關難度階梯表 (30-Level Progression)
難度受 `Rank` (0-255) 與 `StageNumber` 雙重影響。

| 關卡區段 | 難度階段 | 核心特徵描述 | 子彈速率 (px/f) | 敵機 AI 行為 |
| :--- | :---: | :--- | :---: | :--- |
| **Stage 1-2** | 入門 | 基本進場軌跡練習，發彈率極低。 | 1.0 | 緩慢俯衝，不進行軌跡修正。 |
| **Stage 3** | **挑戰** | 第一場挑戰關卡，固定螺旋路徑。 | 0 | 無攻擊性。 |
| **Stage 4-6** | 進階 | 引進 **護衛機 (Guard) 變形** 邏輯。 | 1.2 | 開始出現 2 機協同俯衝。 |
| **Stage 7** | **挑戰** | 第二場挑戰關卡，左右交叉路徑。 | 0 | 無攻擊性。 |
| **Stage 8-10** | 熟練 | 敵機俯衝時會進行小幅度的 X 軸修正。 | 1.5 | 針對玩家 X 座標進行追蹤。 |
| **Stage 11** | **挑戰** | 第三場挑戰關卡，垂直分裂路徑。 | 0 | 無攻擊性。 |
| **Stage 12-18** | 專家 | 敵機發彈率提升 40%，引進快速俯衝。 | 1.8 | 俯衝速度提升至 2.5px/f。 |
| **Stage 19** | **挑戰** | 第四場挑戰關卡，複雜 8 字型路徑。 | 0 | 無攻擊性。 |
| **Stage 20-25** | 大師 | **預測射擊 (Sniper Shot)**：敵機朝預測路徑發彈。 | 2.2 | 敵機俯衝時會突然轉折 (Jerk)。 |
| **Stage 27** | **挑戰** | 第五場挑戰關卡，極速進場。 | 0 | 無攻擊性。 |
| **Stage 28-30** | 極限 | 同時有 3-4 名敵人 Dive，彈幕密集。 | 2.5 | 極高發彈率，玩家需頻繁移動。 |

### 2.1 關鍵關卡節點 (Special Stages)
- **Stage 4 & 5**: 玩家必須在此階段熟悉「變形雜魚」的擊殺時機以獲取高分。
- **Stage 21**: 難度曲線的顯著轉折點 (Spike)，子彈速度首次超過玩家水平移動。

## 3. 五組進場規跡模式 (Entry Trajectory Patterns)
(保留原有的 Batch 1-5 描述，用於每關的起始)

| 批次 | 敵機數量 | 進場位置 | 路徑描述 |
| :--- | :---: | :--- | :--- |
| **第一組** | 4 Boss + 4 Guard | 左上角 1/4 處 | 螺旋向下繞圈後進入編隊。 |
| **第二組** | 8 Grunt | 右上角 1/4 處 | 鏡像第一組軌跡。 |
| **第三組** | 8 Grunt | 左側中段 | 水平進入後轉向上。 |
| **第四組** | 8 Guard | 右側中段 | 水平進入後轉向上。 |
| **第五組** | 8 Grunt | 下方 1/4 處 | U 字型迴轉進入編隊。 |

---
> 🎮 **Design Note**: 30 關之後，遊戲邏輯將進入循環模式，但 Rank 會持續增加直到 255。30 關內的規劃旨在確保玩家能平滑體驗到所有機制的變化。
