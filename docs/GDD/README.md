## 🚀 開發優先序 (Dev Priority)
1. **MVP 階段**: 實現 CPU 1 的進場規跡與 CPU 2 的碰撞偵測。
2. **核心循環**: 完整實作「牽引光束」與「雙機救援」邏輯。
3. **數值調校**: 根據 `04` 文件的 Rank 公式進行難度壓力測試。
4. **細節打磨**: 星空背景的三層視差滾動與音效合成。

---
> ⚠️ **重要提示**: 所有的座標計算應以 288x224 為基準，若需適應現代解析度，應採整數倍縮放以維持像素精確度。
