# GameResearch Project

This project focuses on AI agent-driven research into game design, mechanics, and player experience.

## Objectives
- Use AI Agents to analyze game mechanics.
- Leverage Agent Skills for rapid research and documentation.
- Build innovative game prototypes based on research findings.

## Project Structure
- `.agents/`: Project-specific skills and workflows.
- `docs/`: Research notes, design documents, and case studies.
- `src/`: Codebase for prototypes and agent tools.
- `task.md`: Real-time task tracking for the research.

## Getting Started
To begin research, refer to `task.md` and use the `@mention` system to invoke relevant skills.
